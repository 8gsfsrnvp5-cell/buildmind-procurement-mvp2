# BuildMind Procurement MVP

Первый кликабельный прототип панели контроля закупок для строительного проекта.

## Что уже есть

- данные проекта и даты начала работы;
- страховой запас по времени;
- таблица материалов;
- расчёт свободного остатка;
- расчёт доступного количества;
- расчёт дефицита;
- расчёт даты потребности;
- расчёт крайней даты заказа;
- оценка риска;
- добавление нового материала через форму;
- экспорт данных в JSON.

## Как запустить на компьютере

1. Скачайте ZIP.
2. Распакуйте папку.
3. Откройте файл `index.html` двойным кликом.

## Как загрузить в GitHub

1. Создайте новый репозиторий `buildmind-procurement-mvp`.
2. Нажмите `Add file` → `Upload files`.
3. Загрузите файлы `index.html`, `style.css`, `app.js`, `README.md`.
4. Нажмите `Commit changes`.
